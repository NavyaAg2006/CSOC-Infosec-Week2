# Modular Arithmetic 1
≡ denotes modular congruence and one of its properties is `if a ≡ b mod n`, then `b ≡ a mod n`.  
Thus  
`11 ≡ x mod 6  =>  x ≡ 11 mod 6`  
`8146798528947 ≡ y mod 17  =>  y ≡ 8146798528947 mod 17`  
So we use the following script-
```python
a = int(input())
b = int(input())
print(a%b)
```
